REgEx

RegEx, também conhecido como expressões regulares, é como um filtro de texto. Você pode usar ele para encontrar, substituir ou analisar texto de acordo com um padrão que você define.


Por exemplo, você pode usa-lo para encontrar todos os e-mails em um arquivo de texto. Ou, você pode usar uma expressão regular para validar um número de telefone, foi o que usamos para validar o nosso formulário com o exemplo “/^\(\d{2}\)\d{5}-\d{4}$/” para o telefone.


As expressões regulares são compostas de caracteres especiais e símbolos que representam padrões de caracteres. Por exemplo, a expressão regular `\d` corresponde a qualquer dígito. A expressão regular `[a-z]` corresponde a qualquer letra minúscula.

ALGUNS EXEMPLOS DE EXPRESSÕES REGULARES:

`\w+` corresponde a qualquer sequência de caracteres alfanuméricos, incluindo espaços.

`[a-z0-9_]+` corresponde a qualquer sequência de caracteres alfanuméricos, incluindo sublinhados.

`[0-9]{1,3}` corresponde a uma sequência de 1 a 3 dígitos.

`[a-z][a-z]*` corresponde a uma palavra composta de 1 ou mais letras minúsculas.

* Para encontrar todos os e-mails em um arquivo de texto, você pode usar a expressão regular `\w+@\w+\.\w+`.

* Para validar um número de telefone, você pode usar a expressão regular `\d{10}`.

* Para validar um email '/\S+@\S+\.\S+/'

* Para validar uma data de nascimento /^(0[1-9]|[12][0-9]|3[01])\/(0[1-9]|1[0-2])\/\d{4}$/
